# Instalação 

## Abra o terminal e clone o seguinte repositório:
$ git clone https://github.com/torbenish/Beyou.git

## Instalar as dependências
$ npm install ou $ yarn install

## Iniciar a aplicação
$ npm start ou yarn start

# Tecnologias

- **React**
- **Axios**
- **Vercel**
